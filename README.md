## 📌 Project Overview  
This project presents an interactive **Excel Dashboard** designed to analyze automotive sales data and generate meaningful business insights.  

The dashboard provides a comprehensive view of **sales, profit, mileage, customer performance, and product trends**, helping in data-driven decision-making.

## 🎯 Objectives  

- Analyze overall **sales, profit, and mileage performance**  
- Identify **top-performing brands and models**  
- Understand **customer contribution to revenue**  
- Track **monthly and daily sales trends**  
- Compare **high-value vs normal transactions**  
- Enable interactive filtering for better insights  

---

## 📊 Key Metrics  

- **Total Sales:** 166.8M  
- **Total Profit:** 29,157K  
- **Total Mileage:** 453.7M  
- **Average Car Age:** 10.4 years  
- **Profit Margin:** 21.2%

## 📈 Dashboard Features  

### 🔹 Sales Analysis  
- Month-wise sales trends  
- Day-wise performance insights  
- Seasonal patterns in sales  

### 🔹 Brand & Model Insights  
- Top 5 brands by sales  
- Top 3 brands by profit  
- Top 5 models by sales  
- Units sold analysis  

### 🔹 Customer Insights  
- Top customers by profit and sales  
- Contribution analysis of customers  

### 🔹 Profit Analysis  
- High-value vs normal sales comparison  
- Profit distribution trends
